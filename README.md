# Atomy

사이트를 클론하며 마크업을 연습합니다. 기능은 바닐라JS만을 사용합니다.

## 기능
<ul>
  <li>아코디언 메뉴</li>
  <li>풀페이지 스크롤링, 페이지네이션</li>
  <li>Scroll To Top</li>
  <li>외부 SVG파일 핸들링</li>
  <li>무한루프 슬라이드</li>
  <li>간단한 Module 만들기(3,4page 탭 레이아웃, background.js)</li>
  <li>JSON fetch</li>
  <li>기타</li>
</ul>
