# Atomy

[기업소개 사이트](https://global.atomy.com/)를 클론하며 마크업을 연습합니다. 기능은 바닐라JS만을 사용합니다.

## 기능
+ 아코디언 메뉴
+ 풀페이지 스크롤링, 페이지네이션
+ Scroll To Top
+ 외부 SVG파일 핸들링
+ 무한루프 슬라이드
+ 간단한 Module 만들기(3,4page 탭 레이아웃, background.js)
+ JSON fetch
+ 기타
